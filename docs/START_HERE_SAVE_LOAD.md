## 🚀 How to Use

### Start the App
```bash
cd /home/dobercode/site-builder/frontend
npm start
```

### Create a Project
**Option 1: Blank**
1. Click "New Blank Course"
2. Enter project name
3. Click "Create"
4. Start editing

**Option 2: From Template**
1. Click "From Template"
2. Browse templates
3. Enter course name
4. Click "Create Course"
5. Start customizing

### Auto-Save
- Just start editing
- Changes save automatically
- "✓ Auto-saved" appears
- No manual save needed

### Backup Project
1. Go to Project List
2. Click export icon on project card
3. JSON file downloads
4. Save to your computer

### Restore Backup
1. Go to Project List
2. Click import button
3. Select saved JSON file
4. Project restored and ready to edit
